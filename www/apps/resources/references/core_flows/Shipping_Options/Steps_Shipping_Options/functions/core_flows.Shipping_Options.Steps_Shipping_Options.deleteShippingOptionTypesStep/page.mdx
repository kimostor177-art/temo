# deleteShippingOptionTypesStep - Medusa Core Workflows Reference

This documentation provides a reference to the `deleteShippingOptionTypesStep`. It belongs to the `@medusajs/medusa/core-flows` package.

This step deletes one or more shipping option types.

:::note

This is available starting from [Medusa v2.10.0](https://github.com/medusajs/medusa/releases/tag/v2.10.0).

:::

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@medusajs/framework/workflows-sdk"
import { deleteShippingOptionTypesStep } from "@medusajs/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = deleteShippingOptionTypesStep([
      "{value}"
    ])
  }
)
```
